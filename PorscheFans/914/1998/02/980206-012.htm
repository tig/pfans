It seems to be a fairly common problem, and the repair (though I haven't
done mine yet) doesn't sound all that difficult.  The web page that
describes
the fix is at http://www.estinc.com/porsche/speedo.html, I'd reccommend
looking at http://www.estinc.com/porsche.techref.html if you haven't been
there yet- it's the parent page for the speedo fix and has several other
interesting and useful things listed on it.

If you don't want to fix it yourself, I'd suggest taking it out of the car,
printing the above web page, and taking both to your local speedo
shop- show them the printed web page so have some idea what's
wrong.

Also note that the commonly accepted cause of the odometers mal-
functioning in this way is resetting the trip meter while the car is
moving...
