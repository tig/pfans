At 4:19 PM -0000 2/12/98, Chris Kolstad wrote:
>I have been receiving conflicting reports about what wheel width is legal
>on a Bp 914.  The first time I looked it stated 5.5"+1"=6.5".  The next
>time I looked it stated 5.5"+.5"=6".

	If I recall (not a sure thing by any means!), the Zone 7 rules for
B-production state that you can go to 5.5" + 0.5", or a 6" rim.

> ... But which one is right for both Porsche Club autocrossing and
>Porsche Club road racing? Is there a difference in rim width allowance
>between the two, or should they be using the same rules?  I think it makes
>sense to use the same rules.

	It might make sense for them to use the same set of rules, but they
don't.  As a matter of fact, there are many different rulesets in PCA
driving events.  (Purists will tell you that most of them aren't really
"racing"--and that's what we tell the insurance companies!)  PCA Zone 7
uses a fairly unique set of rules, which are different from those used in
Zone 8, which are different from those used at the PCA National competition,
which are different from those used for Club Racing, which are different
from those used by the SCCA.
	All these groups have different needs to meet, and all have struc-
tured their rules to meet their particular needs.  That can make it tough
to have a car that is competitive in more than one venue, but that's the
way it goes.
