   One other thing, Dave, regarding our conversation about valves.  
Do you have the Publication "How to Rebuild Your Volkswagen
Air-Cooled Engine", by Tom Wilson?   On page 96, there is a good
picture of a typical cam with various areas of the geometry defined,
and on page 97, is an acompanying description of how to measure lift.
  In this discussion, there is a caution that the actual lift cannot
be measured; only the relative lift (comparison between lobes)
because the "base circle" cannot be accurately established.   This is
because the only points available for such a measurement are on the
sides of the lobe, and the "ramp" has already probably started.   
   You need to read Tom's explanation together with his picture, but
unless Porsche wants us to establish valve clearance with the lifter
on the ramp, rather than on the base circle (a practice that I have
never heard of before in any other car), the correct point to measure
valve clearance is with the opposite valve rocking, and not TDC with
both valves closed.   If I interpret the text correctly, the valve
clearance will be greater if set using the TDC method than if set
using the opposite valve rocking technique.   This is not a problem
per se, but the start of the ramp where the clearance is minimal is a
very small part of the cam surface, and it is not likely that you
would get consistant results without a degree wheel on the crank for
reference.   The base circle, on the other hand, comprises the
largest single portion of the cam's surface, and as such, is the
easiest place to get consistant results.
