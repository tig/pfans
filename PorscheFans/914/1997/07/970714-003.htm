Kaefer,

There is no 'best' combination that covers all situations.  Jetting is
largely dependent on your altitude.  And how an engine runs with a
particular setup depends a lot on compression and timing.

However, that aside, I've found that for a stock 2.0, you should have
32mm main Venturis and a good starting point for the main gas jet is
125.  I'm currently running 32mm main Venturis, 125 gas, 185 air, but
I'm planning to get a set of 195 or 200 air to lean it out a bit.

You are actually running the same mixture I am (135/200 = .68, 125/185 =
..68), but you are moving more of the mixture than I am which could be
part of you problem.  I'd recommend as a first step (assuming you have
bigger main Venturis than 28mm which were shipped on a LOT of 40IDFs) is
to change your gas jet to a 125 and give it a try.  If you've got 28mm
Venturis, I'd really recommend bigger ones for a 2.0.
