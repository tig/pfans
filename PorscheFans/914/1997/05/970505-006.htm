Eric, sometimes the metal ball cup on the end of the shaft is so covered with
crap you can't see the small allen head hex key that holds it on.  If you
clean
it well, you'll see it at the bottom of the cup. remove the hex set screw &
it
should come off with some minor persuasion.  if the set screw is broken
or frozen you'll have to drill it out. Good luck.
