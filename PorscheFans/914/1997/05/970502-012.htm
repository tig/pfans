Someone talked of wanting different cams in a 3.0 to get more advantage of 
carbs for HP.  Well, this is true, but you're limited to 964 type 
lift/duration with stock pistons, or Jerry Woods has one similar.  Else the 
pistons get in the way, or so I've been told.

Now, if you opt for forged pistons, heck even oversize 3.2/3.4s, then you 
can machine the tops for valve clearance and put in some radical cams.  :-) 
Course, if you're doing 3.4s, then you might be in the realm of 46 carbs - 
not sure here.
