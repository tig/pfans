Hi gang,

FYI, we need deck clearance (or height) because of carbon build up.  If
you've pulled the heads off before, you'll notice a layer of HARD black
stuff over everything.  

If we run a "zero deck" and no gasket, then we're asking for trouble. 
Carbon build up will make the piston hit the head.    

If you run a head gasket, but "zero deck" then the gasket will give enough
space to keep the piston+head from colliding.
