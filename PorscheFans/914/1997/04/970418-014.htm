O.K., Here's what I've got.  Check my math and logic and maybe we'll all
benefit:

TED = Total Engine Displacement in cc's (2.0l 914 = 1971cc)
DHV = Deck Height Volume in cc's
PDV = Piston Dish Volume in cc's
CCV = Combustion Chamber Volume in cc's

DHV = Deck Height (in inches) x Bore (in inches) x 3.14159 x 16.387
(cc's/cubic inches)

CR = [(TED / No. of Pistons) + CCV + PDV + DHV] / [CCV + PDV + DHV]

Does this look correct?
