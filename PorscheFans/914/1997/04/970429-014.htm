You can buy just the tubing extension from several 914 suppliers. You
could make it yourself using nylon compression tubing and fittings as used
in mechanical gauges. The thread for the pressure switch is an uncommon
one: M10 x 1.00mm, but there is an adapter available between that thread
and a standard US pipe thread, whereupon you can use regular compression
fittings and even run the tubing into the cockpit and have your switch in
there. The only problem with this is that the switch thread needs to be
grounded to the chassis.
