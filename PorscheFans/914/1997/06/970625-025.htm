Maybe I am off base here but from what I understood the tire #'ing was read 
as follows:

205/60-15

205 = Tread width in mm
60 = % of width to height ( so 205/60 is 123mm high (from inter-bead to 
tread)
15 = inch Diameter of inter-bead

Given these #'s you can calculate the rolling diameter of the tire as 
follows:

C = pi x r ^2
205/60-15

       height+15" /2 = R

( ( ( (205/60)/2.54)+15 )/2 )^2) x 3.154

the original tires that came on 914/4 were 165R15  with optional 175s 
(which was a VW match) it was an 80 series tire, but the PCA calculations 
are based on 175/70-15, and no one wants to point it out to rules 
committee. As it is the only tire size that can run legally in stock class 
is 205/60-15. The height variance is .5mm.

As a J stock driver we have been through this deeply wishing there was a 50 
or 55 series tire that fits.
