A suggestion to save time:

In future, please use rationalize all mmfe's (multiple messages for
emphasis).  

For example:

If you wish to send an mmfe message such as
"Porsche #1 had a mid-engine, and so has every *REAL* Porsche since" 
fifty times, this would read
"Porsche #1 had a mid-engine, and so has every *REAL* Porsche
since(50)".

Sending fifty replies to this message would be (50*50) or 
">Porsche #1 had a mid-engine, and so has every *REAL* Porsche since(50)
 >
   Yeah, and your mother uses cinder blocks for jackstands!(2500)"

Please note that the value of a mother using cinder blocks is currently
undefined, so avoid using exponential notation to unless you wish to
have your mme message cancel itself.

Modifications to current mail servers will speed up mmfe reply
generation.  You may have noticed some early beta versions of this
software already beginning to implement this feature, such as AOL.
