This information was in our region's newsletter, provided by Alan Coleman.

Cd = .34 for a 914 with all windows up and targa roof on.
Cd = .39 for a 914 with all windows down and the targa roof on.
Cd = .45 for a 914 with all windows down and targa roof off.
Cd = .43 for a 914 with all windows up and targa roof off.
Cd = 2.0 for a 914 going sideways down the track. ( can D.D. or Holigan atest
to that one ??)

This included driver's side outside mirror and North American ride height (
nose high )
