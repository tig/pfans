First, I agree with Josh that there is no instance that I can think of
(except ice racing) where a NARROWER tire is beneficial.  However, I think
there might be a misunderstanding in Bill's question in that there are
arguements for using a tire of equal width with SHORTER sidewalls.  The
reduced sidewalls increase cornering power, and I seem to recall that a
smaller circumference effectively increases the gearing - which is good for
lower speed driving.  This trick can be used where stock regulations don't
allow changing actual gearing.

Please correct me if I am wrong.
