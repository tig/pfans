If you go this route you want a true relay like the Bosch kit, not a
solenoid.  John Muir mentioned the Ford solenoid in the "Idiots guides" to
VW's.  See the Hoover sermons on the web about why the relay is better than
the solenoid.

The solenoid will work, I've got one on mine from PO so will probably leave
it as long as it works but if just doing it use the relay.
