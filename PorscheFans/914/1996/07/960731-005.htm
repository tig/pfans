They used pneumatic struts like the ones on 911's to hold up the lids  to
raise the headlites.  The cables released a hook that kept the lights down.
 The struts raised the lights.  The headlites had to be lowered manually.
 The cable was actuated by using the truck release handle which was unused in
the GT.
