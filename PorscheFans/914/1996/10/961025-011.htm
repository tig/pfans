CYLINDER HEAD TEMPERATURE PARAMETERS OF PORSCHE 
914

Dear Mr. Thorusen,

During the model year 1970 up to model year 1976 we built the 
vehicles of the type 914 in cooperation with the Volkswagen AG.

In those years we equipped the four cylinder vehicles with engines 
made by Volkswagen and the six cylinder vehicles with engines made 
by Porsche.

Since we never had any problems concerning extended cylinder head 
temperature we did not find necessary to publish technical data to this 
topic in our workshop manual.

For checking the temperature sensor II, which is situated close to the 
spark plug for the 3rd. cylinder, you will have to work in the following 
procedure:

Disconnect plug, connect ohmmeter between temperature sensor II 
and ground, read indication.

Nominal value:   0.5 - 2.5 kilohms (very temperature sensitive, value 
decreases with increasing temperature).

Guide value       approx. 2.5 kilohms when engine temperature is 
approx. 20 degrees C (68 degrees F).

Replace unit if necessary.


Kind regards,
