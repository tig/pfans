Andy:  Here's how I understand it....I think...... 
 
Stock cars are allowed suspension changes as outlined in the rules. Engine 
mods are limited to anything before the intake(air filter) and after the 
exhaust ports. That's it.  If you change the motor by anything more than a 
rebuild with .5 compression rebuild, it's now "prepared".  It moves up a class 
and has to still make stock minimum weight.  
 
If you up the displacement or make any other changes beyond the .5 increase in 
compression, it goes to a GT class based on displacement. 
 
If you lighten the car and it doesn't make minimum stock weight, it goes into 
a GT class where there is no(presently) weight minimum. If minimun weights are 
imposed in the GT classes, you would have to make minimum weight for the class. 
 
This car sounds like a GT2 class car......I think it's up to 3.2 liters for 
GT2.  Someone will no doubt correct me if I'm wrong.  Don't have the rule book 
with me.....
