My problem is excessive tire wear during club time trialing.  The car is a
completely stock '86 951 using R1's, with front alignment set at 0 toe and -1.5
degrees camber.  This setup worked well for autocrossing, but not so well on
the track.  Apparently during hard braking, the front geometry changes towards
more positive toe and more negative camber, which is causing some rather
aggressive scrub wear on the inside of the front tires.  I intend to dial in
some negative toe, but would welcome any recommendations on more suitable
alignment settings.
