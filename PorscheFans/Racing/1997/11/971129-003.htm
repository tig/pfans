I do believe SCCA IT rules allow for a welded in cage.  Showroom Stock
does not.  As far as ITE goes, it is true that it is a regional only
class, but not an official SCCA regional class.  The regions that run
that class set the rules that they want.  At most, you might find
similar rules within a division.  I'm not as active in SCCA as I once
was, but I don't think ITE is run in the Southeast Division.  Such as
the case, if I run my PCA GT4S car, it will have to run SCCA GT2.  At
the regional level, I should be OK, depending on who else shows up.
At the National level, I'd be toast.
