I use Nitrous on my street rod. NOS would be effective on the 928, however,
stay below 150 hp. The 90 - 125 hp kit would be perfect. The engine would
not take more especially if you did not add a separate fuel cell and fuel
pump to guarantee adequate fuel supply. The cost is about $350 - $400 U.S. 
The kit is easy to install if you purchase the smaller unit. The only
problem with nitrous is that it only provides boost at full throttle and
only for a few seconds. During those few seconds it would be very
impressive.
