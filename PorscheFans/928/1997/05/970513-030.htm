Ok...

WPO	World mfr code for Porsche  
JBO 	filler symbol or 'vehicle specific code for US cars'
92 	first two digits of 928
2	test code or filler symbol
K	1989 model
S	place of production - Stuttgart!
8	third digit of 928
6	code for body and engine 
1206	# (0001 to 1250 - no breakout given S4 vs. GT)

I believe we're looking for code 81L00501 to  02591 which is the numbering
for the '90 GT engines.   But you know Porsche, it's entirely possible they
used 'S4' blocks (81K00051-00629) for the first run of GT's...

source:  The Porsche Family Tree

So, the ten million dollar question is what is your engine number!!!  Other
telltale signs of a GT?  Sport suspension (does your car ride like a truck
but corner on rails).  The most obvious is 'Club Sport' wheels (7-spoke
wagon wheel lookin' things).
