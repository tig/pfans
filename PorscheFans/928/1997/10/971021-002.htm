Last year my 84 928 would just die for no reason. Would run great for a few 
miles, then sputter and die.  Cranking it over would not get as much as a 
sputter.  After it would sit for 20-40 minutes, it would start up and run 
fine.

The dealer replaced the fuel pump/filer/relay, then the Fuel pressure 
regulators, but no change.  Poor or no diagnostics by the tech.  I expected 
more from a dealer.

The culprit ended up being the computer "Brain", a $1,000 part.  I found 
used and rebuilt ones around for $600, but they only had 30-90 day 
warranties. For that much, I wanted a better warranty, so I coughed up the 
extra $ for a new genuine Porsche part.

The brain controls the FI system and when it starts to go, one of the 
symptoms is it shuts down the FI system intermittently.  Acts as if some 
electronic component gets hot and shorts out.

I think this could be tested if one were to hook up a volt meter and check 
for a pulsing signal at the injectors while cranking the engine. There is 
probably another to test it in print somewhere, but the dealer I went to 
did not seem to know about it.

Bill Smyth
'84 928S
