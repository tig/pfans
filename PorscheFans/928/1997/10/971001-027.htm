 One way to find out if you have an LSD is to pull out of a driveway or street
making a left or right turn and PUNCH IT. If you smoke the inside  (of the turn)
tire you do not have an LSD . If you wind up in the ditch on the other side of
the road, congratulation you do have an LSD. ;-) You can amplify this test by
doing it on a rainy day .

On a serious note Code 221 is for the later "active" PSD. My understanding is
this was standard in MY90, even though standard it still had an option code,
strange.

Code M220 is for the non-active limited slip which has a built in 40% slip
ratio, you can look at the number before the year code letter on the tranny
serial number, 1 is without LSD, 2 is with LSD.
For example: Trans serial # G2855 2 K 0001 indicates a G28/55 type trans with
limited slip for MY 89 serial #1.

Code M221 is for the "active" hydraulically controlled differential , which in
reality is not a limited slip. This system uses the ABS sensors and an
additional acceleration sensor to automatically adjust the power from one wheel
to the other. It is indefinitely variable from 0-100%.
