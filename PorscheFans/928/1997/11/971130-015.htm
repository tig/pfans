Alex:

This may be a shot in the dark,  but sometimes the simple and abvious
aren't.  Reading the owner's manual for my '89,  there's a section that
talks about the security system.  According to that,  tampering with the car
(like trying to start it with the security engaged) will cause the system to
activate, disabling the engine.  Perhaps the system thinks your clifford
starter is trying to 'hotwire' the car, and the system is activating to
"protect" the car.

So:  Try locking the car with the key, then unlocking with the key to
disable the security system.  With luck, you will be able to start the car
now with the ignition key.  Look at yopur owner's manual to see if I am
talking through my hat-- it's happened maybe twice before...
