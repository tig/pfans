>Is there any reason I should not put synthetic oil (Mobil 1 )
>in the engine

No, there isn't. You'll get all kinds of benefits; the only drawback is a
few more bucks per oil change, which you can write off as insurance.

>and synthetic lube ( AMSoil ) in the transaxle?

If that transaxle contains a manual transmission, you may not want to go
full synthetic therein. The synchros appear to need some friction to work
properly, and some gear lubes are too good at lubricating them, leading to
stiff shifting and graunching. Try if you want, but be ready to revert.
