> Ok. What is MIME format, and why do my messages always appear with it?

Probably because you are using Microsoft Outlook Express. These headers 
can be found in your message:

X-msmail-priority: Normal
X-mailer: Microsoft Outlook Express 4.72.2106.4
X-mimeole: Produced By Microsoft MimeOLE V4.72.2106.4

If memory serves, geno@flinet.com (Gene O'Rourke) was bit by the MIME 
problem and finally broke down and figured out how to cure it once and 
for all and volunteered to share his cures.

FWIW: MIME is a good thing. My messages are in MIME format too. But I 
only use the "Content-Type: text/plain" unless enclosing an attachment 
and then the message part is still text/plain.
