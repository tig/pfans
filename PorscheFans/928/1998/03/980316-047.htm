Brian,

If you don't know where the cruise control is, how
can you be so sure there is no vacuum leak?

Did you test the line that goes directly to the
vacuum store _for_ the cruise control? You can't have
because if you'd tested that line then you'd know
where the control was. 

Beware, there are seperate systems for the heating
controls, cruise and possibly (depending on MY) the
door locks. All of these systems incorporate check
valves, so leaks in one system shouldn't be able to
be detected in another system (unless the check valve
is fuct as well).

The CC unit is behind the splash guard of the inner
fender on the driver's side of the car, rear of the
wheel/tire, in front of the door hinges. Fun to get to.
