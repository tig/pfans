	Did they bother to check the coolant temperature sensor to see if it is within
spec?  Also check for a broken circuit to this temp sensor.  This is NOT the sensor that
is used for the temperature gauge.  If I recollect, the LH system uses the coolant
temperature sensor for initial idle speed.  This is done to give enough time for the O2
sensor to heat up and kick in.  Once the O2 sensor is up, it is then used to fine tune the
idle speed.  There is also a throttle switch that works in conjunction with the throttle
body and the air mass meter.  If any one of these component is out of calibration (or
malfunction..) it will show the symptom that you are experiencing.  I can be mistaken
since I am recalling all this from a distant past....
