  I've completed the first test release of the search engine. 
Try it out on my personal web page at

  http://users.abac.com/jbronte/ArchiveSearch.html

  I'll be putting a package together to submit to the list and
web masters. I think you'll find it useful once it's installed
permanently in the PorscheFans web pages.

  A few notes:

  - I have only two weeks of 928 only messages in the test archive,
    so trying another list will return a blank and more than one
    month will be ignored.

  - Multiple keywords are matched as 'AND' in the whole message,
    that is, "fuel injectors" matches the words "fuel" and "injectors"
    separately, but both must be present in the same message to hit.

  - For long or numerous hits, wait until the result has read all
    the way in, because if you select a reference, it will abort the
    download and you won't get all your hits.

  Comments are appreciated, especially any error conditions you
encounter. TIA.
