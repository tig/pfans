For starters, I would certainly put in a fresh set of WR7DC spark plugs. They
are optional with the 5's, but if you had no symptom prior to replacing the
plugs, this would be a logical first step. Second, it may sound funny, but
run a coupla tanks of mid grade (good quality) gas thru the car instead of
super premium. I have seen this fix several twin cam cars (and C2's)  with no
other symptom other than cold start / run complaints. .. Hope this helps...
