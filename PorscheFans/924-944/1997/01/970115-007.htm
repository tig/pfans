My '85 944 used to exhibit the same symptoms. It would be running great and
then it would want to die. Sometimes I could keep it alive; other times it
would ve dead. After a while it would run again.

This only happened in the coldest of weather.

I finally concluded that it was icing and cured the problem with dry gas.

YMMV, but in my case after I started using dry gas in the winter I never had
the problem again.

NOTE: My current 944S2 has never had the problem
