Mine does exactly this same thing if I use anything less than 20w-50 oil. If
you are already using this weight oil, try replacing the sender first
(easy). If that doesn't do the trick, have the gauge calibrated (not too
expensive, but check quotes against the cost of a new gauge). Finally, if it
is none of these, learn to blow taps and say a silent prayer over your bank
book. :-)
