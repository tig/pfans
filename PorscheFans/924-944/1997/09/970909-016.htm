Hmmm... so you want a funky Porsche screensaver.

The best system I have on my computer is AVI movie files of the 1996-1997 
Porsche factory series of ads narrated by Captain Picard.

To do this you need the following;

+ AVI Screensaver (beta version/shareware version) found at either 
WWW.SHAREWARE.com or WWW.HOTFILES.com (ZNET). It is a relatively small and 
fast program to download.

+ once you have this, you can run any AVI type file as a screensaver. To 
adjust the AVI Screensaver options, simply go into the WINDOWS95 
'properties' screen like you normally would to add a screensaver, and you 
will see the AVI options.

+ search the net for Porsche related AVI files (the ads I found were from a 
German site... I can't remember where... sorry)

+ once you've found what you want, download the AVI, set the options in 
'properties', and enjoy.

If you need more instructions, email me at JARNOLD@fzsi.com and I will try 
to help.
