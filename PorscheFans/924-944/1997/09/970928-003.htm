 Hmmm, there seems to be a lot of disinformation in this note...

The two sensors both feed info to the DME, a failure in either one will =
keep the engine from running
at all. They do have an intermittent failure mode, which will cause =
intermittent running problems.

The DME relay can be bypassed with the appropriate jumpers.

For more accurate information, read the 944 FAQ...
