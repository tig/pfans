Section 1, Purchasing:

*       If it is perfect, someting is overlooked.

*       There is yet another fault.

*       A failure will not appear until guarantee is void. =


*       The better alternative emerges only after a deal is closed.

*       Hidden flaws will not stay hidden.

*       If things feels just right, something bad is bound to happen.

*       If it doesn't feel quite right, that observation is correct.

        embarrassing location.

*       The other Porsche is quicker.

*       Donsbachs Delusion : Subjective acceleration is a direct
        function of how far you have to reach out the side window =

        to see where you're going.

*       A speed related Police situation will immediately go from =

        bad to worse as the Police reads 't-u-r-b-o'.

*       If driving starts to be fun, your girlfriend gets sick.

*       The amount of traveling pleasure derived from both front =

        seats is constant.

*       If it is a nice day for a drive, you get stuck in a traffic jam.

*       Ruudjes Observation of Traffic Intensity: If there wasn't any
        traffic at all, you missed the game of the year.

*       Spencers Theorem : If there is a beautiful, twisty stretch of
        road ahead, there will also be Winnebagos.

*       Spencers Second Corrolary: If there are neither Winnebagos or
