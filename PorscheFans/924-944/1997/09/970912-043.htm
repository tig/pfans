Greg, yes it is a good idea to replace both at one time.  They both see the
same number of in/out cycles.  It should save on labor costs since the
system only needs to be bled once.  Also, it is not that difficult to
change these if you own a good socket set and extensions (and some line
wrenches).  New OEM parts can be bought for about $100 from Vertex in Miami
(total for both parts).
