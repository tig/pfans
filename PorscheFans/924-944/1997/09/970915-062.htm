 As far as I know forged Fuchs will not work stock (without 
modifications) on 87' and newer 944's, because of the changed offset 
(52.3mm) for the ABS system. 
Remember that there are several different versions of the Fuchs (widths, 
offsets). Beware! Try before you buy! I am not sure which will work with 
spacers. Any input from the list on this one?
