I don't have the generic numbers handy.  I do regularly buy generic 
bearings from local shops and haven't found their prices to be any 
different from the ones you'll get from Automotion, Imparts, etc. -- 
Burton Malkiel was right.

If you want to go this route, simply pull your current bearings one 
morning, run down to the bearing shop, and they should be able to read 
the numbers off the bearings and pull replacements off the shelf.  I've 
done this for my own late-model wheel bearings.
