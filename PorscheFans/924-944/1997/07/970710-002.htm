>I've sent You an e-mail massage a week ago, asking if You can check
>my Porsche 924's chassis number for me (WPOZZZ92ZDN404222), and
>tell me something more about the car(production year etc.)

WPO		= world manufacturer code; PO = Porsche
ZZZ		= only a filler (ZZZ) on european (non-US) cars
92		= first two digits of 924
Z		= filler
D		= model year = 1983 (E=1984..., 'I' and 'O' were left out)
N		= car built in (N)eckarsulm (location of Audi)
4		= last digit of 924
0		= body|engine code
4222	= serial number

>Also, I've found a sticker in the spare wheel compartement with some
>strange numbers:

>Options: 000  440  568  931

440 = manual antenna
568 = tinted windshield and side glass
931 = seat cover rear KKK (whatever that means...)
