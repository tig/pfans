Your stock radio should have a diagram of the plugs and the pinouts for 


the power connector on a label underneath the unit (mine does).

Staring with the wider spades on the factory connector, it is (bad ASCII 
art, sorry):

	|  |  |  |
	|  |

ground (earth), 12 volts switched, (next spade, not sure, check with a
test light or meter - I think it is the power antenna control, if
equipped), last one is 12 volts unswitched (for memory). 

You might consider the plastic bin piece to go underneath the radio - 
handy storage space instead of a blank panel.
