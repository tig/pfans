Gewneral FYI on 245/275 combo:

If you go 17" you need to be careful.  Noone, as far as I know, makes
a low cost "off the rack" wheel for the 944turbo to run 245/40+275/40/17.

The recommended wheel sizes (according to BFG) are 8.5 + 9.5-10 x17
The only 1-peice wheels available aftermarket are 7.5+9.0x17, which is 
on the narrow size for these tires.

That means you go into "custom built" land (read $$$$) in getting this
combo fo the car.

As far as I know pricing on the correct wheels new is approx $2K to 
start (HRE).
