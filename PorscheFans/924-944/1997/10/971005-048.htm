Dear Rick & P-Fans

	I would probably suspect a bad battery. If you have a maint. free type
battery (sealed - no screw caps) this type of battery can only be discharge
once or twice before it is toast! The maint. type battery (with screw caps)
can be discharged many times 4-6 times with out killing the battery's ability
to hold a charge. If you battery is still covered by a warranty - get it
replaced! 
	I would also check for any electrical drains on the system. An easy way to
do so is to use a test lamp (12 Volts). Remove the battery's ground wire and
attach the test lamp between the negative post and the battery's  negative
wire. The test lamp should only be dimly lit - not bright! If it is bright
you have something in the electrical systems staying on! Make sure everything
is switch off, and start pulling fuses - one by one, to see which circuit the
short on!
