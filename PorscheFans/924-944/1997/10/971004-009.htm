Rick -

Sounds like you need to get a dc amp meter and see if there is any draw on
the battery when everything is off (key, radio, etc.).  This meter is put
in-line (in series, not parellel) between positive lead and the battery
positive post (it can also be put in-line with the negative cable but then
the meter leads need to be reversed).  If there is no draw on the battery,
then the battery is gone.  Otherwise, you need to find where the draw is
coming from by using this meter again (with battery connected) across each
fuse (with fuse removed) to see where the shorted or draining culprit is
located.  Then you can fix it from there.  There are a couple of things 
like the clock and any radio memory back-up that will always draw current 
form the battery but I would venture to guess that these draws are in the 
10-25 milliamp range and very negligeable on the battery.  Now something 
in the 100-150 milliamp range will kill a battery in about a week or 
two.  Of course, something higher would kill it sooner.
