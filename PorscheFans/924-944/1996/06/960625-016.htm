William wrote

>I have an early '85 944 and the test pipe is detached from the main exhaust
>pipe.  I had a muffler shop weld it back on three different times but it
>always break off after a short time.  Anyone out there have this problem
>and how do you remedy this without replacing the entire exhaust pipe?

You need to make sure the shop is using the right type of weld wire.  The
mufler is stainless, not sure which type.  Also, the weld wire should be a
low carbon stainless.  If you are using an Cr/Ni wire the code should have
an 'L' in it (308L).  The 'L' stands for low carbon.  If the system is Cr
only (409) you might want to try a flux core wire.  The grain refinement in
the weld is much better than many of the soild wires.

Went through this on a friends headers until the shop realized they needed
to use the right wire.

Hope this helps
