NO NO NO!  Dot 5 is NOT compatible with DOT 3 or 4.  Do not for any reason
attempt to combine Dot 3 and/or 4 with DOT 5.  To use DOT 5 silicone you
must completely drain all of the old fluid out (assuming it is a DOT 3 or 4)
before using the DOT 5.  I have heard some recommend that you even change
the ALL the seals and reservoir before using DOT 5.

Just trying to prevent an expensive lesson for someone.
