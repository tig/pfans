OK - so they may be old hat to many, but in case anyone cares...hereit is...

Most wheels, generally measure 1/2"-3/4" wider to the lip since the rim 
width measurement is to the inside bead where the tire contacts.

Having both measurements would be most helpful.

If you can imagine this as a wheel crossection:

_ <----C------> _
 |_____________|
      | |
      | |
      | |
      | ||

<- A ----|-- B-->    (Inside)
                    
      | ||    < ----(Hub Flange)
      | |
      | |
  ____| |_____
_|            |_       


In the diagram above "C" is the wheel width

A - B
-----  = offset  (i.e. distance to th ehub flange from the wheel centerline
A + B

and 

A+B > C

Hopefully my feable attempt at computer graphics was not in vain....

BTW;  Standard offsets are as follows:

911/930 -10.8 or -23mm (911 7 or 8" are 10.8, 9" are 23, so they put the entire 1" to the inside)
944 through '86  -23mm
944 late/C2/993  -53 mm
928  -53 or -60mm 
993tt rear 40mm.
