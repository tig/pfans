Regarding the Sears battery thread, all batteries are date coded. It's
usually a format like "C96" meaning March '96 or some variant of that. I
never buy a battery unless the code shows it to be fresh. I bought my
944's battery from Walmart for $47. So far so good. There are only a few
battery manufacturers in the country that produce all of the batteries
no matter how many different names are on them.
