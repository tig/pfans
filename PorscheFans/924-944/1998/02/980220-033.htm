I had the same problem when I had my S2. Sounds like you are only going to
do fairly basic repairs, so what I would recommend is buying the standard
944 manual.  About 80% of the stuff is the same and was quite helpful a few
times, so for the $15 it is worth it.  I am currently trying to use a
standard '68-'88 911 manual for my '89 C4 which is a completely different
car, but the basic components still tend to be similar so it helps, sometimes.
