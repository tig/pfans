Need help with an '82 924 starting problem. The engine starts normally,
runs about 20-30 seconds then dies. Instantly restarts, runs 20-30 seconds,
dies again. 

Leaving it at idle or running it up to about 2,000 rpm during 20-30 seconds
it will run has no effect on run time. Running the rpm's up does seem to
reduce the number of restarts needed to get it to keep running. Warms
engine faster?

Number of times restart required, until engine will continue to run, is
temperature dependant. With a cold engine and outside temperature above 90
degrees it will run on first start after a slight stumble. At 70 degrees it
may require 2-3 restarts. At 50 degrees it will need 5-6 restarts before it
will continue to run.

After driving, when engine is warm, no starting problems at all.

Any ideas of where to look?
