Non-S turbos were available with oil cooler and limited slip, but these
were not standard.  Type 016R, Code 5P had the cooler, and Code 5R had
both cooler and LSD.  Type 016S, Code UY had no cooler or LSD, and Code =
9U
had no cooler, but LSD.

So that means the only real difference for the S box is blasting (some
type of shot peening) of 1st and 2nd gears, which is not really the same
as hardening, which implies heat treatment.  If you are doing an overhaul
you can do this at the same time.

>Also, regular diffs wear out.  They use a friction surface (like a =

>brake) to transfer the torque to the wheel that's not spinning.

slippery" formula.
