Thore,
 It is hard to believe, but if you remove "shift pin" as you called it, you 
will see substantial wear. At some point, it will get so bad that you will not 
be able to properly shift. You can rig a bandaid fix until you obtain a new 
part by using some metal tape to build up the diameter of the pin. I used 
stainless steel tape for a couple of weeks while waiting for a replacement, it 
helped a lot..

Good Luck!
