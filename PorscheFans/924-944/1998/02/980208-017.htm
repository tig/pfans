I have used Mobil 1 synthetic 75W90 for a number of years with fine 
results.  I am sure that Redline is OK too - just be sure to use the 
right variant if you have an LSD.

To test for an LSD, jack both rear wheels off the ground, car in 
neutral.  Spin one wheel by hand.  If the other spins the same 
direction, LSD, else open diff.
