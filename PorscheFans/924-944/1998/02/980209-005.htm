	A preface here: I do not know exactly how the Porsche system works but I
would bet it is not entirely different than the system VW/Audi use which I am
aquainted with. 
	A knock sensor DOES NEED to hear "some" backround noise in-oder for the
"knock sensor" to opperate properly. If you do not buy into this - simply
disconnect the knock sensor and drive the car. You will notice a huge HP loss.
The knock sensor is a tuned pizo electric crystal, in-order for the ECU to
work properly, the ECU looks for this "back-round" noise. If the knock sensor
bolt attaching the sensor to the engine, is not torqued down (too loose). Or
if the sensor has been "crushed " sensor will be damage. You must replace the
sensor in that case.  I believe the only way to test a sensor is to measure
the resistance, not the voltage. I dod not know the exact resistance specs -
though. For some reason, VW/Audi/Porsche all test components with the engine
NOT running (most all other manufacters, test components while the engine IS
running).
