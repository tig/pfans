One suggestion for using keywords in the subject line: 

If the post is Non-Technical, start the post with Ntech, then subject. For
example, the recent string on Drivers Education would be:  Ntech-Drivers Ed

If the post was technical, I would recommend starting the post with the car
type, then the system to which the post relates. For example, the recent
string on turbo boost would be:  951-turbo boost, and the string on the
GReddy control unit would be: 951-turbo boost-GReddy. If the message was a
reply, the word "reply" could follow the subject.

Any comments or other ideas?
