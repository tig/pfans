Please note that I did not neglect the cold start up factor in my post. 
Also, if you feel that the CU test is meaningless to anyone but fleet
operators, I would welcome anyone to share similar scientific,
statistically supportable data that does include the cold start factor.  I
have seen nothing that says a 3K oil change interval will necessarily
decrease cold start up wear (from oil starvation) as compared to a longer
change interval.  If cold start up is in fact the prime cause of engine
wear, then it seems to me that much more is to be gained by switching to
synthetic than to using arbitrarily short oil change intervals.
