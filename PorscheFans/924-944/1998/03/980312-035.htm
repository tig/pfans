Ezra, the manual does indeed state that old (used?) 574 doesn't need to be
removed.  I would recommend it though, old loctite 574 is hard and may decrease
tolerances if it is left on.  An example I've experienced is opening up the oil
pump.  When I put it together I didn't remove all the old 574, so the back
didn't go flush with the back of the pump housing.  (so I re-opened it and
cleaned it.
