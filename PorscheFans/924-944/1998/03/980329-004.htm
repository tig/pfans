The Lambda sensor compensates for some changes in the engine.  If a
valve leaks or there is an intake leak, the lambda sensor brings the
system back into design limits.  If the sensor is disconnected, the
control unit operates at a programmed injection rate.  Some service
procedures will no longer be reliable since they depend on the
sensor being warm and sending a signal.  The sensor continually
adjust the air-fuel ratio by the fuel-injector pulse time. I suggest
it's best to fix the sensor.
