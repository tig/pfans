My UK '88 944 Turbo has the following VIN :-  WPOZZZ95ZJN100564.
I understand the characters to mean :- 

WPO     World mfr code  stands for "West Germany - Porsche"  
ZZZ     Filler/vehicle specific code (VDS code) - ZZZ for Euro car, 
95      Car type (95=Turbo, 94=non-Turbo)
Z       Test digit
JN      Model year: JN - 1988.
100564  Serial number 

There is no place of manufacture character. I thought that from 1981
there was an international standard VIN with the following layout :-

1981 International VIN System - 17 Characters

1-3     World manufacturer code; WPO = Porsche
4-6     VDS code
7-8     Model type
9       Test digit
10      Model year 
11      Manufacture; S = Stuttgart, N = Neckarsulm
12      Additional type data
13      Body/engine code
14-17   Serial number

This differs from my 944 VIN and from previous USA 944 VINs posted. Does
anyone know why this is so please?
