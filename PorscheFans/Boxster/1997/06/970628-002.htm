-- Actually, the "remote" version of the Valentine is a misnomer.  For
an extra fee, you can buy a module that contains all the warning
lights.  This option connects to the main unit via a cable,
deactivitates the lights on the main unit, and allows you to place the
light module somewhere it can't be seen from the outside.  However, for
the main unit to continue to work well, especially the side and rear
sensing features, it has to be mounted up, usually in the windshield. 
They need a true remote version, much like another manufacturer, that
has a front and rear sensor that gets mounted on the ends of the car
with a warning module inside somewhere.
