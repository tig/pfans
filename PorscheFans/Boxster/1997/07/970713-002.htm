Someone has just written regarding the check engine light.  I had posted that
problem earlier on this list stating that the culprit was a balky oxygen sensor
that needed to be replaced twice.  The computer analyzer showed several
indications of the problem.  Since the second O2 sensor no more problems.  

The problem orignated about 150 miles into a cruising trip both times it
occured.  Hope this helps.  This particular problem is not fatal.  The apparent
only possible down side is that the on board diagnostics might think something
is really wrong and adjust the timing, mixture, etc. to compensate.
