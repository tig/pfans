According to the "Service Information Technik" the numbering is:

WPO ZZZ 98 Z V S 6 00001...10000 ROW
WPO CA2 98 x V S 3 20001...30000 USA
-------   -----  --   -  -  -   -  -------------------
    |          |      |    |  |  |    |             |
    |          |      |    |  |  |    |             ->Serial number (1st digit ==> Code for Body)
    |          |      |    |  |  |    ->Vehicle Type with 7th and 8th digit
    |          |      |    |  |  -> Production Site (Stuttgart)
    |          |      |    |  -> Model Year
    |          |      |    -> ROW Fillwe/USA number code
    |          |      -> Vehile type with 12th digit
    |          -> ROW Filler/USA Code for body/enginetype/restraint system
    -> World Manufacturing code

So if Porsche sticks to usual form, either just the Production Site will differ, and/or the World Manufacture 
Code. As an example of what can happen, in Europe the VWs just have different PS codes, even in different 
countries, yet the Mexican cars have different WMC. Go figure!
