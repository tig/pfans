I think you have the 'complete' list.
My list shows P17 is the code for manual trans, and P18 as the code for Tiptronic, otherwise the 
same package. BTW, I believe these have been renumbered P43/P44. The "Technic Sport 
package" (P38) is the tuned springs, shocks, sway bars, traction control and 17" Boxster-style 
wheels/tires. Be aware of the P16 "Sport Package" which has only the wheels, none of the other 
suspension bits (lots of other stuff though).
