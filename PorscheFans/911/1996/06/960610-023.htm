Jim,
According to my Stoddard's guide; #9113102819 is possibly a'73 T, 2.4liter
CIS coupe possibly (originally) fitted with a sporto.  The sequence starts
at 9113101501 for CIS cars, but no final serial number is listed for this
run and the next run (for the '73 E models) starts at 9113200001,  and the
'73 S models start at 911330001...no, this number does not belong to a
targa, as those sequences start with #911311xxx, as opposedto
911310xxx.(and so on...)
So, if this thing is a targa...something's hosed up...

I got all this out of the back of Stoddard's  911 catalogue...so, if you
find out that this is erroneous data for whatever reason -please tell me
and I'll cease to refer to it.

Hope this helps
