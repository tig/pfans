My only suggestion is to add some more safety gear. If you haven't
already done so, you really should add a set of 5 or 6 point harnesses.

to modify the car that much.  You can add the harness without any
drilling or cutting, and it will make a BIG difference if something bad
happens at Mid Ohio.
