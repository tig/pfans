Gearbox number 7331137 is a '73 915/02.
It should be a 5 sp gearbox. According to a Porsche Technical spec. book
it should have the following gear ratios
1st 11:35, 2ed 18:33, 3rd 23:29, 4th 26:25, 5th 29:22 with a  ring
and pinion of 7:31
Also each gear set should have a stamped 2 letter gear code along with
hand etched matching numbers. The letter code can be used th identify 
the gear ratios. Or you can count the teeth.
