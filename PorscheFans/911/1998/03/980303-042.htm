Wayne,

An oxygen sensor is an oxygen sensor.  They all use the same chemistry.
The generic ones are just the same as the brand names.  The only difference is
between 2-wire and 3-wire sensors.  3-wire sensors have an internal heater
(the sensor needs to be about 600 deg F or so to work---I could be wrong
about the exact number, but it's high---and the 3-wire guys heat
themselves, while the 2-wire guys need to be situated close to the exhaust
manifold to get hot enough.

If you can find a generic sensor that fits, don't hestitate to use it.
