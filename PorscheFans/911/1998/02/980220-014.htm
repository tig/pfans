"David J. Kessler"  asked :

>> I'm going to be autocrossing my '88 Carrera for the first time this 
weekend and I was wondering if 
anyone has recommendations as to what tire pressures I should start 
with. 
Here's the specifics: bone stock '88 Carrera narrow body, Comp T/A ZR 
225/50 on 7x15 Fuchs on front, Comp T/A ZR 245/50 on 8x15 Fuchs on 
rear. Should I just go with the factory pressures (28F/36R if memory 
serves) +5 psi or is there a better starting point? All suggestions 
are welcome. TIA. <<

Probably start at 35 ft 38 rear and go from there depending
on tire's roll over and car's balance. More air in a comp ta ZR 
will help that end stick better, up to a point (~mid 40s). 
The higher (relative to factory) front pressure will help 
minimize the dreaded understeer.

Good luck!
