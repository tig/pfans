My opinion on 2.7's ( after owning one for about 6 years ) is they are
great IF the head studs have been replaced and the case has had threaded
inserts installed. Current thinking on head studs is to use the Raceware
items, not the Dilivar.

Heat was the big enemy of these motors. Put an early exhaust on them (
SSI's are great ) with an OEM type muffler ( less problems than aftermarket
types ). If you are in a hot area a better front oil cooler may have to be
used, or added if you don't already have one.

As you can see they can be quite expensive to fix properly. If has already
been done - great, if not maybe a 3.0 SC would be better in the long run.
