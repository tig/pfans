Pfans:
I've noted some discussion on tensioner update procedures as of late.   
 The information given was fine and I would only add the following words   
of caution.  Set the engine at #1 TDC before you start and make sure you   
match mark the cam sprocket to the cover.  Wiring the chains together is   
fine but if they jump they are more likely to jump at the crank sprocket   
and with the match mark you'll see it and can correct it before doing any   
harm.  Happy wrenching.
