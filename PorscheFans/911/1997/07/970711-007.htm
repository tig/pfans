As to the wire shielded sets, I would trash them. If you feel RF
interference is a factor then I would make your own set and use an RF
suppression wire with a low resistance. Make sure it is has a solid wire in
the core somewhere ( usually spiral wound around a core ). Your plug ends
also have to be the type with the wood screw wire end.

Boughten sets ( $ 80 ) are the best deal if you don't feel RF concerns are
valid.
