Re Dave Kessler's question about '74 and later steering wheel:   The
horn pad attaches via a snap on each corner and just pulls off.  The
whole thing is molded plastic, with the contacts molded in.  It's
possible that it might be on too tight to remove without breakage, if
it's never been off, so I'd shoot some WD40 into each corner.  The
technique is to pull evenly so that no corner is unduly stressed.
