I've not tinkered with Zeniths, but have Solexes and Webers; all of
these being fixed venturi, progression type carbs.  Seems to me that
float level is not the problem since even the normal level is above all
of the progression holes.  Given this, there must be a check valve to
prevent the fuel from running out.  On the Solex I believe the check
valve is part of the idle jet carrier.  If this is also the case for the
Zenith, you might try swapping the idle jet to see if this makes a
difference. Also. maybe the larger idle jet is the problem.  I would be
interested in what you find, so please e-mail.  Greg
