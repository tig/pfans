Phil.

I've been worrying about this for my '65 911.  I haven't put new tires on
yet, so don't go off believing my advice about fit is expirimentally
confirmed.

The height of a 165HR15 Michelin XAS tire is 25.6 inches (see
http://www.coker.com ).  They are available, but cost about $ 180 each with
tubes.  Coker also sells a 185/70HR15 Michelin XWXT, which is 25.5 inches
high (I think it costs even more).

Since you (and I) are looking for tires with the same diameter, here are
the nominal, or calculated diameters of a couple tire sizes from Dunlop's
website (http://www.dunloptire.com):

Dunlop D40 M2	(Z and V rated)			
			Rim	Tire	Tread	
Size			Width	Diam.	Width	
195/65R1591V		6.5	25.1	6.5	
P205/65R15 92V	6.5	25.6	6.8	
P205/60R15 90V	6.5	24.8	7	
P215/60ZR15		7	25.2	7.3	


Dunlop D60 A2	(H rated)			
			Rim	Tire	Tread	
Size			Width	Diam.	Width	
P185/65R15 86H	5	24.5	5.51	
P195/65R15 89H	5.5	25.0	5.91	
P205/65R15 92H	5.5	25.5	6.06	
P215/65R15 95H	6	26.1	6.46	
P195/60R15 87H	5.5	24.1	6.22	
P205/60R15 90H	5.5	24.6	6.44	
P215/60R15 93H	6	25.2	6.86	

As you can see, a 205/65R15 is about as close in diameter to the original
165HR15 tires as we can get.  However, I've been told that a 195 width tire
is about as wide as an early Porsche can handle.  Sinc a 1969 car should
have some fender flare, maybe you could change the wheel offset slightly
and go to a 205.  That takes either new wheels (expensive) or spacers.

I have seen a couple 912s running on 195/60R15 tires, so that size should
be safe.
